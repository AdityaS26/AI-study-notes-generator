
# AI Study Notes Generator (GenAI Project)

This project generates study notes, key points, summaries, and exam questions from a given topic using a Generative AI model.

## Technologies Used
- Python
- Streamlit
- Hugging Face Transformers
- FLAN-T5 model

## How to Run

1. Install dependencies

pip install -r requirements.txt

2. Run the application

streamlit run app.py

3. Open the browser link shown in the terminal.

## Example

Input:
Database Normalization

Output:
- Explanation
- Key points
- Summary
- Important exam questions
